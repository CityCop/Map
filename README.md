* Node 10.8.0
* Git

## Common setup

Clone the repo and install the dependencies.

```bash
git clone ....
cd crimeometer-visualizations
```

```bash
npm install
```

To start node.js server, run the following

```bash
npm start
```

Open [http://localhost:8080](http://localhost:8080) and take a look around.
